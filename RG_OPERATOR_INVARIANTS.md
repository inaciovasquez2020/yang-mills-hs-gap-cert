RG Operator Invariants

Core structural invariants required for spectral gap:

1. RG Intertwiner
   U : H_phys(Λ) → H_phys(Λ')
   Condition: [U, L] = 0

2. OS Positivity Preservation
   e^{-tH} D_OS ⊂ D_OS

3. Uniform Mourre Estimate
   i[H, A] ≥ c0 L − C0 1, with c0 > 0 independent of cutoff

4. Violation Exclusion
   No sequence ψ_n with L(ψ_n)=1 and <ψ_n,H ψ_n> → 0

Mass gap follows if and only if all four hold nonperturbatively.
