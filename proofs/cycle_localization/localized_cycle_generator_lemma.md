# Localized Cycle Generator Lemma

Let \(\gamma\) be a cycle in the lattice gauge complex.

There exists a generator \(\Phi(\gamma)\) satisfying

\[
\partial \Phi(\gamma) = \gamma
\]

and

\[
\operatorname{supp}(\Phi(\gamma)) \subset N_R(\gamma)
\]

for some radius \(R = O(1)\) independent of the lattice volume.

Interpretation:

- gauge constraints allow cycle excitations to be realized by localized generators
- excitation support grows only by a bounded neighborhood of the cycle
- this prevents long-range hidden fluctuation modes.

This lemma ensures that cycle fluctuations belong to the localized subspace
\(V_{\mathrm{cycle}}\).
