# Local Filling Operator

Let \(\gamma \in \ker(\partial_1)\) be a cycle.

A filling operator is a map

\[
\Phi : \ker(\partial_1) \to C_2
\]

such that

\[
\partial_2 \Phi(\gamma) = \gamma.
\]

The constructive goal is to choose \(\Phi(\gamma)\) using only plaquettes in a bounded neighborhood of the support of \(\gamma\).
