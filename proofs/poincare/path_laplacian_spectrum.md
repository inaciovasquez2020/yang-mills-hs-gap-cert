# Path Graph Laplacian Spectrum

Let \(P_L\) be the path graph on vertices \(\{1,\dots,L\}\) with Dirichlet boundary.

Eigenfunctions:

\[
\phi_k(x) = \sin\left(\frac{\pi k x}{L+1}\right), \qquad 1 \le k \le L
\]

Eigenvalues:

\[
\lambda_{P_L}(k) = 2 - 2\cos\left(\frac{\pi k}{L+1}\right)
\]

Hence

\[
\lambda_1(P_L) = 2 - 2\cos\left(\frac{\pi}{L+1}\right).
\]
