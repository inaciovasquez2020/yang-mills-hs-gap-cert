# Analytic \(L^{-2}\) Lower Bound for the Cube Gap

From the product-spectrum formula,

\[
\lambda_1(B_L)
=
2 - 2\cos\!\left(\frac{\pi}{L+1}\right).
\]

Using the bound

\[
1 - \cos x \ge \frac{x^2}{4}
\qquad (0 \le x \le \pi),
\]

we obtain

\[
\lambda_1(B_L)
\ge
\frac{\pi^2}{2(L+1)^2}.
\]

Since \((L+1)^2 \le 4L^2\) for \(L \ge 1\),

\[
\lambda_1(B_L)
\ge
\frac{\pi^2}{8} L^{-2}.
\]

Thus there exists \(c>0\) such that

\[
\lambda_1(B_L) \ge c L^{-2}.
\]
