# Cube Spectral Gap from Tensor Decomposition

Using the tensor-product spectrum,

\[
\lambda(k_1,\dots,k_d)
=
\sum_{i=1}^{d}
\left(2 - 2\cos\frac{\pi k_i}{L+1}\right).
\]

The smallest nonzero eigenvalue occurs when one coordinate equals \(1\) and the rest equal \(0\).

Thus

\[
\lambda_1(B_L^{(d)}) = 2 - 2\cos\left(\frac{\pi}{L+1}\right).
\]

Using the bound

\[
1-\cos x \ge \frac{x^2}{4}
\]

gives

\[
\lambda_1(B_L^{(d)}) \ge \frac{\pi^2}{2(L+1)^2}.
\]

Hence

\[
\lambda_1(B_L^{(d)}) \ge cL^{-2}.
\]
