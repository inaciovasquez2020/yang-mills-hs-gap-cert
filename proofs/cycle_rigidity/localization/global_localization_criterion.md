# Global Localization Criterion

Let G be a bounded-degree block-intersection graph.

A family of blocks or cycle supports is admissible only if there exists v such that

support(Φ(γ)) ⊂ B_R(v).

This is strictly stronger than:
1. bounded-radius local homogeneity
2. pairwise overlap diameter bounds
3. consecutive block intersections

Empirical correction:
local cycle-rank bounds apply only to globally localized cycle families.
