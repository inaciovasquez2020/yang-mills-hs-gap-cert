# Final Localized Cycle-Rigidity Bound

Assumptions

1. Global localization:
   supp(Φ(γ)) ⊂ B_R(v)

2. Ball cycle-space bound:
   dim Z1(B_R(v)) ≤ f(Δ,R)

3. Interval-Helly reduction provides a center v for admissible localized supports.

Conclusion

Every admissible cycle class lies in the local graph B_R(v), hence

C_R(G) ≤ dim Z1(B_R(v)) ≤ f(Δ,R).

Using the established ball estimate,

f(Δ,R) = O(Δ^(R+1)).

Therefore

C_R(G) ≤ O(Δ^(R+1)).
