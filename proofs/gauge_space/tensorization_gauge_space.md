# Tensorization in Gauge Configuration Space

Assume each plaquette sector carries a local Laplacian \(L_p\) with uniform gap

\[
\lambda_1(L_p)\ge c_0>0.
\]

For a block of linear size \(L\), the block configuration space \(\mathcal A_L\) is a tensor product of local plaquette sectors.

Under block averaging, the first nonzero block mode has wavelength \(L\), hence

\[
\lambda_1(\mathcal A_L)\ge cL^{-2}
\]

for some \(c>0\).

This is the gauge-space analogue of the cube spectral-gap estimate.
