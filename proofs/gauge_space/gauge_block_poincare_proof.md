# Gauge-Space Block Poincaré Proof Layer

Let \(P_L\) denote block averaging on gauge configuration space and let

\[
Q_L=I-P_L.
\]

Let the gauge Dirichlet form be

\[
\mathcal E(f)=\sum_p \langle \partial_{U_p}f,\partial_{U_p}f\rangle.
\]

If

\[
\lambda_1(\mathcal A_L)\ge cL^{-2},
\]

then by the Rayleigh characterization,

\[
\|Q_L f\|^2
\le
\lambda_1(\mathcal A_L)^{-1}\mathcal E(f)
\le
CL^2\mathcal E(f).
\]

Thus

\[
\|Q_L f\|^2 \le C L^2 \mathcal E(f).
\]
