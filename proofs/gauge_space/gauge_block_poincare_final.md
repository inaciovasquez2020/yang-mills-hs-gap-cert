# Gauge Block Poincaré Inequality (Final Layer)

Let \(P_L\) denote block averaging on gauge configuration space and

\[
Q_L = I - P_L.
\]

Using the Rayleigh characterization and the block spectral estimate

\[
\lambda_1(\mathcal A_L) \ge cL^{-2},
\]

we obtain

\[
\|Q_L f\|^2
\le
\lambda_1(\mathcal A_L)^{-1}\mathcal E(f)
\le
CL^2 \mathcal E(f).
\]

Therefore the gauge-space block Poincaré inequality holds.
