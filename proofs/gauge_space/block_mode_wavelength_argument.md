# Block Mode Wavelength Argument

Block modes represent fluctuations with wavelength proportional to the block size.

For a block of linear scale \(L\), the first non-constant mode has wavelength \(L\).

The corresponding Laplacian eigenvalue scales as

\[
\lambda_1(\text{block mode}) \sim L^{-2}.
\]

Thus

\[
\lambda_1(\mathcal A_L) \ge c L^{-2}.
\]
