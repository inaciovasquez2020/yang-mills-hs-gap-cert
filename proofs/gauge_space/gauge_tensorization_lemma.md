# Gauge Tensorization Lemma

Let the gauge configuration block \(\mathcal A_L\) contain \(m(L)\) plaquette sectors.

Assume each local plaquette Laplacian \(L_p\) satisfies

\[
\lambda_1(L_p) \ge c_0 > 0.
\]

The block configuration space is the tensor product of local sectors.

The block Dirichlet form decomposes

\[
\mathcal E(f) = \sum_p \|\partial_{U_p} f\|^2.
\]

Tensor-product coercivity implies

\[
\lambda_1(\mathcal A_L)
=
\min_{f\perp 1}
\frac{\mathcal E(f)}{\|f\|^2}
\ge c_0 \lambda_1(\text{block modes}).
\]
