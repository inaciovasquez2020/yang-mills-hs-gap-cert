# Defect Measure Control

## Sequence
Let \(f_L\) satisfy
\[
\|f_L\|=1,\qquad
\sum_{\ell}|\nabla_\ell f_L|^2 \le C L^{-2}.
\]

## Defect measure
Define
\[
\mu_L := \sum_{\ell} |\nabla_\ell f_L|^2 \,\delta_\ell.
\]

## Total mass
\[
\mu_L(\text{all edges}) \le C L^{-2}.
\]

## Rescaled measure
Under block scaling,
\[
\tilde{\mu}_L := L^2 \mu_L.
\]

## Uniform bound
\[
\tilde{\mu}_L(\cdot) \le C.
\]

## Limit measure
\[
\tilde{\mu}_L \rightharpoonup \mu.
\]

## Rigidity
\[
\mu = 0 \;\Rightarrow\; \nabla g = 0.
\]

## Consequence
All limit mass collapses; no nontrivial fluctuation survives.

## Status
Defect measure enforces vanishing-gradient limit.
