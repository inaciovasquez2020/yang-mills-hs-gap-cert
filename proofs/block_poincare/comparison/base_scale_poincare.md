# Base-Scale Poincaré Inequality

## Block
Let \(B\) be a block of side length \(L\).

## Mean-zero reduction
\[
\sum_{x\in B} f(x) = 0.
\]

## Discrete Poincaré (finite block)
\[
\sum_{x\in B} |f(x)|^2 \le C L^2 \sum_{\ell \subset B} |\nabla_\ell f|^2.
\]

## Spectral form
\[
\lambda_1(-\Delta_B) \ge c L^{-2}.
\]

## Normalization
\[
\|f\|^2 \le C L^2 \mathcal{E}_B(f,f).
\]

## Status
Finite-block spectral gap; base case for multiscale induction.
