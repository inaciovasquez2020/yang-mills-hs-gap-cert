# Constant Extraction and Uniformity

## Combined inequality
\[
\|Q_L f\|^2 \le C L^2 \sum_{\ell} |\nabla_\ell f|^2.
\]

## Scaling normalization
Define
\[
c_L := L^{-2}\frac{\|Q_L f\|^2}{\sum_{\ell}|\nabla_\ell f|^2}.
\]

## Empirical stabilization
\[
c_L \to c \in (0,\infty).
\]

## Uniform bound
\[
c_L \le C \quad \forall L.
\]

## Target lower bound
\[
c_L \ge c_0 > 0.
\]

## Cyclone condition
\[
\exists\, c_0>0:\quad
\|Q_L f\|^2 \ge c_0 L^2 \sum_{\ell}|\nabla_\ell f|^2.
\]

## Status
Upper bound closed; lower bound (Cyclone) remains.
