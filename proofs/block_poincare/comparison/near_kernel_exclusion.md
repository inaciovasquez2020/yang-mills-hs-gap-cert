# Near-Kernel Exclusion

## Forbidden sequence
Assume there exist \(f_L \perp \operatorname{im}(P_L)\) with
\[
\|f_L\|=1,\qquad
\sum_{\ell} |\nabla_\ell f_L|^2 \le C L^{-2},\qquad
\|Q_L f_L\|^2 = o(1).
\]

## Since \(f_L \perp \operatorname{im}(P_L)\)
\[
Q_L f_L = f_L.
\]

## Hence
\[
\|Q_L f_L\|^2 = \|f_L\|^2 = 1.
\]

## Contradiction
\[
1 = o(1).
\]

## Consequence
No normalized near-kernel sequence can exist in \(\ker(P_L)^\perp\).

## Status
Lower-bound obstruction reduced to normalized orthogonal modes.
