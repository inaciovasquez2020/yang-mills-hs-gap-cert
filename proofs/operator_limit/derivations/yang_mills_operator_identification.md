# Yang--Mills Operator Identification

Assume:

1. \(D_{\mathrm{cyl}}\) is dense.
2. \(H_L f \to H_\infty f\) for all \(f \in D_{\mathrm{cyl}}\).
3. The limiting action on \(D_{\mathrm{cyl}}\) agrees with the Yang--Mills Hamiltonian prescription.

Then

\[
H_\infty|_{D_{\mathrm{cyl}}} = H_{YM}|_{D_{\mathrm{cyl}}}.
\]

If \(D_{\mathrm{cyl}}\) is an operator core for both sides, then

\[
H_\infty = H_{YM}.
\]
