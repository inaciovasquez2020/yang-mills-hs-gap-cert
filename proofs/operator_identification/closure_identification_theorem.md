# Closure Identification Theorem

Let \(H_L\) be finite-volume Hamiltonians and assume

\[
H_L \xrightarrow{s.r.} H_\infty .
\]

Let \(D_{\mathrm{cyl}}\) be the cylinder-function subspace.

Assume

\[
D_{\mathrm{cyl}}\subset D(H_\infty)\cap D(H_{YM}),
\]

\[
H_L f = H_{YM} f
\quad
\text{for all } f\in D_{\mathrm{cyl}}
\text{ and sufficiently large }L,
\]

and

\[
D_{\mathrm{cyl}}
\text{ is a graph core for } H_{YM}.
\]

Then

\[
H_\infty|_{D_{\mathrm{cyl}}} = H_{YM}|_{D_{\mathrm{cyl}}}.
\]

Since \(D_{\mathrm{cyl}}\) is a graph core,

\[
\overline{H_{YM}|_{D_{\mathrm{cyl}}}} = H_{YM}.
\]

Because \(H_\infty\) is self-adjoint,

\[
H_\infty = \overline{H_\infty|_{D_{\mathrm{cyl}}}}.
\]

Therefore

\[
H_\infty = H_{YM}.
\]
