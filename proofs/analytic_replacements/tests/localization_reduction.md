# Localization Reduction

Given projected interval families I_i^(k):

1. Pairwise overlap implies total overlap in one dimension.
2. Choose c_k in intersection_i I_i^(k).
3. Set v = (c_1,...,c_d).
4. Then every support point lies within coordinate distance <= R from v.
5. Hence supp(Phi(gamma)) subset B_R(v).

Remaining sublemma:
projection of admissible block supports preserves the overlap hypotheses.
