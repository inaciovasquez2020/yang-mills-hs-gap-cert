# Operator Form of Block Poincaré Inequality

Using the cube spectral gap,

\[
\lambda_1(B_L) \ge cL^{-2}.
\]

For functions orthogonal to constants,

\[
\|f\|^2 \le \lambda_1(B_L)^{-1}\langle f,(-\Delta)f\rangle.
\]

Applying this to \(Q_L f\) gives

\[
\|Q_L f\|^2 \le C L^2 \langle f,(-\Delta)f\rangle.
\]
