# Cycle Support Localization

Let \(\gamma\) be a cycle excitation.

Define the cycle excitation operator \(\Phi(\gamma)\).

Locality of plaquette interactions implies

\[
\operatorname{supp}(\Phi(\gamma)) \subseteq N_r(\gamma)
\]

where \(N_r(\gamma)\) is the radius-\(r\) neighborhood of the cycle.

Since each plaquette term is local,

\[
r = O(1).
\]

Thus

\[
\operatorname{supp}(\Phi(\gamma)) \subset B_R(v)
\]

for some \(R\) depending only on the cycle diameter.
