# Block Spectral Gap Core Lemma

Status: OPEN

Canonical target:

For block size \(L\) and blocked action \(S_B\),
\[
\sup_U \left\|\nabla^2 S_B(U)-\beta \Delta_B\right\| \le C L^{-2}.
\]

Equivalent spectral form:
\[
\gamma_B \ge c L^{-2}.
\]

Role:
This is the single open core lemma required by the conditional Yang--Mills mass-gap route in this repository.

Branch rule:
