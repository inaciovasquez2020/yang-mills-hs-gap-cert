\begin{abstract}
This note presents a conditional structural framework connecting operator-growth
bounds (Pulse) to locality collapse in four-dimensional Yang--Mills theory.
The framework is explicitly informed by computer-assisted verification results
certifying the failure of Hilbert--Schmidt and Kato-type coercivity routes.
No numerical computation is claimed to establish the Yang--Mills mass gap.
\end{abstract}

\end{remark}
