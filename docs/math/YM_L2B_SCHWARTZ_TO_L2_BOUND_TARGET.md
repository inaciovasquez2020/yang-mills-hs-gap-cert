# L2b Schwartz-to-\(\ell^2\) Bound Target

## Status

Conditional.

## Target

\[
\mathbf{(L2b)}\quad
\sum_{x\in\Lambda_n}|K_{n,f}(x)|^2\le C\|f\|_{\mathcal S,m}^2.
\]

Using
\[
K_{n,f}(x)=(W_n f)(x),
\]
this becomes
\[
\sum_{x\in\Lambda_n}|(W_n f)(x)|^2\le C\|f\|_{\mathcal S,m}^2.
\]

## Weakest sufficient formulation

There exist \(C>0\) and \(m\in\mathbb N\) such that for every \(n\),
\[
\|W_n f\|_{\ell^2(\Lambda_n;\mathfrak g)}\le C\|f\|_{\mathcal S,m}
\qquad
\forall f\in\mathcal S(\mathbb R^4,\mathfrak g).
\]

Equivalently,
\[
W_n:\mathcal S(\mathbb R^4,\mathfrak g)\to \ell^2(\Lambda_n;\mathfrak g)
\]
is uniformly bounded in \(n\) with respect to one fixed Schwartz seminorm.

## Reduction lock

\[
\text{uniform Schwartz-to-}\ell^2\text{ bound for }W_n
\Longrightarrow
\mathbf{(L2b)}.
\]

## Next admissible micro-fix

\[
\boxed{\text{write }W_n\text{ as a sampling/averaging operator and bound it by one Schwartz seminorm}}
\]
