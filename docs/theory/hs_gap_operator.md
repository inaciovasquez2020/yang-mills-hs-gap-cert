# Hilbert–Schmidt Gap Operator

Let T be the transfer operator induced by one RG blocking step.

Define the Hilbert–Schmidt operator

K = T^* T

acting on L^2 gauge configurations.

The mass-gap certificate requires

λ₂(K) ≤ (1 - γ)

for some γ > 0 independent of lattice size.

This implies exponential decay of correlations.
