# Schatten-p Witness Layer

Let T be the normalized Yang-Mills operator in the HS-gap framework.

For p > 2 define the finite-spectrum Schatten witness

W_p(N) = \left( \sum_{j=1}^N s_j(T)^p \right)^{1/p}

where s_j(T) are the singular values of T restricted to the first N certified modes.

Purpose:

1. quantify stronger-norm finite-spectrum decay,
2. separate HS divergence from higher-p finite truncation behavior,
3. produce non-claiming numerical witnesses consistent with the frozen negative result.

Interpretation:

- HS obstruction may persist globally;
- finite-spectrum Schatten-p decay can still be measured;
- no mass-gap claim follows from this witness alone.
