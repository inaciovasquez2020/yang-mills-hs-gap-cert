\section*{Small-$R$ Limit Analysis}

Define

\[
E_0(R)
=
\inf \mathrm{supp}(\mu_R).
\]

Let

\[
m_* = \liminf_{R\to 0} E_0(R).
\]

Question:

\[
m_{\text{gap}} = m_* ?
\]

Obstacle:

$\Pi_R$ introduces external static sources,
while glueball states contain no sources.

A source-decoupling theorem would be required.
